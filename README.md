## 📁 폴더 규칙
- 최상위는 **`번호_이름`** 폴더 (예: `001_HTML`, `002_AI automation`) — Finder에서 한눈에 식별
- 번호는 **수업 시작 순서대로 그때그때 추가** (001, 002, 003 ...)
- 각 번호 폴더 안에 실제 작업 폴더:
  - `bootstrap/` → 연습용 샌드박스 (교재 Bootstrap)
  - `project/` → 제출용 과제 (교재 Project)
- 공통 폴더: `notes/` (일자별 노트), `resources/` (참고자료), `main-project/` (누적 신문 프로젝트)
- bootstrap/project 안의 폴더·파일명은 영어 kebab-case (터미널 친화적)
