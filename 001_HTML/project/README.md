# The Campus Chronicle

A simple front-end prototype for the school's digital newspaper, built with
plain HTML5 and CSS3 (no frameworks).

## What's inside

- `index.html` — the newspaper homepage (header, hero story, article grid, footer)
- `styles.css` — all the styling

## How to open

Just open `index.html` in any browser. Double-click the file, or run:

```
open index.html
```

No build step or server needed.
