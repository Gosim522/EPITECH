- 태그는 항상 닫기: `<p>...</p>` (슬래시 `/` 빠뜨리면 깨짐)
- 중첩하면 Tab 들여쓰기 (부모-자식 관계 가시화)
- `<head>` = 안 보이는 정보 / `<body>` = 화면에 보이는 내용
